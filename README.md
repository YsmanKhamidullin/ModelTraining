### Getting start

Streamlit app:

```
1. Execute: streamlit run streamlit_app.py
App will open browser itself
```

FastApi app:

```
1. Execute: uvicorn fastapi_app:app --reload
2. Open in browser: http://127.0.0.1:8000/docs#/
```
