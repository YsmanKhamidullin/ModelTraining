# Translating text from screen to screen
### What and Why
This project is for translating the text in your screen.

### Getting start
1. Download project
[Download last Release](https://null)
2. Execute main.py:
```
python3 main.py
```

### Current Implemetation target
Current implementation oriented for:
Japan -> English
Japan -> Russian
English -> Russian
